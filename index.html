Crear tareas

Descripción

Facilita la creación de tareas del proceso en Azure

Instalación

# Descargar el instalador: Installer.zip
# Descomprimir y ejecutar el archivo AzureTasksSetup.msi
# Instalar en la ruta que se desee, por ejemplo: C:\Program Files\Mayansoft\Azure task creator\
# Clonar el repositorio

git clone  https://github.com/fernandoCab26/AzureTasks.git

Uso

            
 # Ejemplo de uso
 Ingresar a la carpeta de instalacción, por ejemplo: C:\Program Files\Mayansoft\Azure task creator\

 Ubicar el archivo Configuration.json y agregar las organizaciones en las que se requieran agregar tareas.

 Si es necesario, modificar las tareas o agregar o eliminar las que se consideren necesarias para el equipo.

                    
 #Importante

 Cada vez que se actualice el archivo configuration.json se debe reiniciar la aplicación.
