Crear tareas

Descripción

Facilita la creación de tareas del proceso en Azure

Instalación

# Descargar el instalador: Installer.zip
# Descomprimir y ejecutar el archivo AzureTasks.UI.exe

Uso

            
# Ejemplo de uso

Se debe generar un token en Azure Devops con permisos read,write y manage para Work Items

En el archivo Configuration.json se deberá definir los valores de la organización , proyecto , equipo, token  y las tareas del proceso que aplican para el proyecto.

#Importante

Cada vez que se actualice el archivo configuration.json se debe reiniciar la aplicación.
